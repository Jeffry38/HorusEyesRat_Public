* Hide & Show Desktop Icons

* Tasks Manager : Kill , Resume , Pause

* History Recovery (+35 web browsers based on chromium)
* Wifi Passwords Recovery
* Power : Log out , Reboot , Shutdown , Hibernate , Suspend
